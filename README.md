# The </intuinet>

The internet, if GUI's **never existed**

## Information

The Intuinet is a hobbyist project made by me (me), and
aims to be an alternative to most terminal browsers today.
Currently, The Intuinet is more of a "fancy terminal document
renderer" than an actual web

## Documentation

**╖ `<h>text</h>`**
║ A header, Makes the text inside
║ it large. There are currently
║ no plans to make diferent
║ sizes of `<h>`.
 
**╖ `<p>text</p>`**
║ Normal text, Rendered as normal.
 
**╖ `<bold>`, `<ital>`, `<under>`, `<strike>`, and it's closing tags**
║ Turns the text inside into bold,
║ italic, underlined, or strike-through.
 
**╖ `<fg;col=FFFFFF>text</fg>`**
║ Sets the text color to the
║ HEX color inside of the tag.
║ Note: MUST HAVE NO HASHTAG.
 
**╖ `<bg;col=FFFFFF>text</bg>`**
║ Sets the background color to the
║ HEX color inside of the tag.
║ Note: MUST HAVE NO HASHTAG.
 
**╖ `<raw>text</raw>`**
║ Sets the background color to the
║ HEX color inside of the tag.
║ Note: MUST HAVE NO HASHTAG.
